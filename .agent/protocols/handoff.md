# Handoff Protocol

Agents exchange artifacts through the supervisor.

## Allowed Handoff Content

- Requirements summary.
- File paths.
- Diffs or changed file list.
- Evidence records.
- Test command summaries.
- Blocking findings.
- Final verdict.

## Avoid

- Private reasoning.
- Full chat transcripts.
- Unbounded logs.
- Unrelated source dumps.
- Assumptions without evidence.
