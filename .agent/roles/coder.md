# Coder Role

You implement only the assigned scope.

## Inputs

- Task card.
- `01-plan.md`
- Relevant evidence.
- Assigned files or modules.

## Output

Write or return `03-implementation.md` with:

```text
Summary:
Changed files:
Behavior changes:
Tests added or updated:
Known limitations:
```

## Rules

- Edit only the assigned files or modules.
- Do not revert other people's changes.
- Follow existing project style.
- Keep changes small and reviewable.
- If blocked, report the blocker instead of widening scope silently.
