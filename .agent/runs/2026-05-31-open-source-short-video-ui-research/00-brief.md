# Brief

## Request

Research GitHub open-source Flutter frontends suitable for the short-drama interactive demo.

## Evaluation Criteria

- Flutter-based.
- Short-video / TikTok / Reels / vertical video feed relevance.
- Clear permissive license.
- Easy to modify and integrate with current local player features.
- Not strongly coupled to backend/auth where possible.

## Non-goals

- Clone or import a repository in this turn.
- Change app code.
