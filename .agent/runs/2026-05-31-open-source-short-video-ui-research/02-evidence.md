- `FlutterWiz/flutter_video_feed`: MIT, explicitly targets TikTok/Reels/Shorts-style video feed, MVVM structure, LRU video caching.
- `salvadordeveloper/flutter-tiktok`: MIT, TikTok UI and swipe videos, but old and Firebase-coupled.
- `nostrlabs-io/freeflow`: MIT, complete TikTok-like app, but coupled to Nostr, ObjectBox, wallet, relays.
- `pablin202/tiktok_clone_flutter`: focused vertical videos, but no clear license found in GitHub UI.
- `PrimeTimeTran/flutter-TikTok`: useful UI reference, no clear license found in GitHub UI.
- `devgony/tiktok_clone`: useful tutorial repo, Firebase/functions/node_modules coupling; no clear license found in GitHub UI.
- `itsezlife/flutter-instagram-offline-first-clone`: MIT and current, but too broad/heavy for short-drama v0.3 shell.
- `feed_player`: API looks relevant, but pub.dev reports unknown license and unverified uploader.
